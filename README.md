# Excel-HR-Analytics-Dashboard

🚀 # Project Overview

This project demonstrates how Microsoft Excel can be used to build a professional HR Analytics Dashboard that analyzes employee attrition trends. It highlights skills in data cleaning, pivot table analysis, visualization, and reporting.

The dashboard consolidates workforce data into interactive charts and KPIs, enabling HR teams to monitor retention, identify risk factors, and design targeted engagement strategies.

📈 # Key Metrics

Total Employees: 1470

Attrition: 237

Active Employees: 1233

Attrition Rate: 16.12%

Average Age: 37 years

Job Satisfaction Rating: 2.6

🔍 Attrition Analysis

By Gender: Female (588), Male (882)

By Education: Bachelor’s (99), Master’s (58), Associate’s (44), High School (31), Doctoral (5)

By Department: HR (5.06%), R&D (38.62%), Sales (56.12%)

By Age Group: Under 25 (38), 25–34 (112), 35–44 (51), 45–54 (25), Over 55 (11)

By Marital Status: Single (120), Married (84), Divorced (33)

💡 Insights & Recommendations

Sales Department: Highest attrition (56%). Requires retention-focused initiatives.

Early-Career Employees: 25–34 age group shows highest turnover. Mentorship programs recommended.

Job Satisfaction: Average rating of 2.6 suggests moderate dissatisfaction. Engagement initiatives needed.

Education Trends: Bachelor’s degree holders show highest attrition. Upskilling opportunities may help.

⚙️ Technical Implementation (Excel)

Data Collection

Imported raw HR dataset into Excel (.xlsx/.csv).

Verified column headers and checked for missing values manually.

Data Cleaning

Removed duplicates using Excel’s Remove Duplicates feature.

Standardized categorical values (e.g., “Sales Exec” → “Sales Executive”).

Ensured numeric fields (Age, Attrition Count) were properly formatted.

Data Validation

Used pivot tables to cross-check totals (e.g., attrition + active = total employees).

Reconciled employee counts across gender, department, and education categories.

Dashboard Creation

Built pivot tables for attrition by department, education, age group, marital status.

Designed charts (bar, pie, KPI cards) for visualization.

Added slicers for Education and Department filters.

Formatted dashboard layout for clarity and recruiter-facing presentation.

Reporting

Highlighted KPIs: attrition rate, active employees, job satisfaction.

Generated insights for HR decision-making.

🛠️ Usage

Download the Excel file from dashboard/.

Open in Microsoft Excel.

Use slicers to filter by Education and Department.

Explore attrition trends interactively.

📌 Conclusion

This HR Analytics Dashboard shows how Excel alone can deliver professional workforce insights. It highlights strong skills in data cleaning, validation, and visualization, making it a valuable project for showcasing analytical capabilities.
